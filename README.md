# Scalin - Langage Fonctionnel & Objet Inspiré de Scala

Langage de programmation fonctionnel pur et purement objet, inspiré de Scala.
Compilateur en C++ générant du NASM x86_64.

## Structure
- src/ - Code source du compilateur
- include/ - Headers
- tests/ - Tests unitaires
- examples/ - Exemples Scalin
